# CoinCap Portfolio (React + Redux Toolkit + Ant Design)

Веб-приложение для просмотра криптовалют (CoinCap API), просмотра деталей монеты с графиком и ведения пользовательского портфеля (покупка/удаление). Портфель сохраняется в localStorage и не теряется после перезагрузки страницы.

Возможности приложения: есть таблица со списком монет и их статистикой (rank, symbol, price, market cap, change и т.д.), данные обновляются автоматически примерно каждые 30 секунд, доступен переход в детали монеты (информация + график истории), покупка монеты выполняется через модальное окно по кнопке “+”, а также есть портфель пользователя: отображается общая стоимость портфеля в USD, отображается разница относительно изначальной стоимости (USD и %), открывается модальное окно портфеля со списком монет, количеством, текущей ценой и итогом по строке, и есть возможность удалить монету из портфеля. Портфель сохраняется между перезапусками приложения с помощью redux-persist.

Используемые технологии: React, React Router, Redux Toolkit, redux-persist, Ant Design, Axios, Vite.

Требования: Node.js 18+ (рекомендуется) и пакетный менеджер npm/yarn/pnpm.

Установка и запуск: в терминале выполнить команду npm install, затем npm run dev. После запуска открыть адрес из консоли (обычно http://localhost:5173).

Переменные окружения (.env): в корне проекта рядом с package.json нужно создать файл .env и добавить строку VITE_TOKEN_COINCAP=ваш_токен. После изменения .env необходимо перезапустить dev-сервер.

Как пользоваться приложением. На главной странице “/” вверху отображаются 3 популярные криптовалюты (топ-3 по rank) и информация о портфеле, ниже находится таблица со всеми монетами. Данные автоматически обновляются примерно каждые 30 секунд, а время последнего обновления отображается над таблицей. Чтобы купить монету из таблицы, нужно нажать кнопку “+” в строке выбранной монеты, в модальном окне ввести количество и нажать “Купить”, после чего монета добавится в портфель и итоговая стоимость портфеля пересчитается. Страница деталей монеты открывается по адресу “/coin/:slug” при клике по строке в таблице, на ней отображается подробная информация и график, а справа над карточками находится кнопка “+” для покупки монеты с текущей страницы через модалку. В портфеле вверху справа отображается “Итого”: текущая стоимость портфеля и разница (USD и процент). При клике на блок портфеля открывается модальное окно со списком активов, в котором можно удалить любую монету кнопкой “Убрать”.

Возможные проблемы и решения. Если приложение не загружает данные и в консоли видно, что токен не найден, проверь что файл .env лежит в корне проекта рядом с package.json, переменная названа строго с префиксом VITE_ (например VITE_TOKEN_COINCAP=...), и после любых изменений .env ты перезапустил dev-сервер (остановить и снова выполнить npm run dev). Если не отображаются иконки Ant Design или появляются ошибки импорта иконок, установи пакет командой npm i @ant-design/icons и перезапусти dev-сервер. Если после установки пакетов или обновлений Vite ругается на кеш зависимостей, можно удалить кеш папку node_modules/.vite и снова запустить проект, либо просто переустановить зависимости (удалить node_modules и выполнить npm install). Если npm сообщает о уязвимостях (npm audit), можно выполнить npm audit fix, но если после этого что-то сломалось, лучше откатить изменения и фиксить зависимости аккуратно.
